## Warum Prism?

Jedes Mal, wenn Sie ein neues Gespräch mit einem KI-Codierassistenten beginnen, startet er von Null. Sie erklären die Architektur erneut, beschreiben Entscheidungen erneut, listen TODOs erneut auf. Stunden an Kontext — verloren.

**Prism gibt Ihrem Agenten ein Gehirn, das bestehen bleibt — und lehrt ihn dann zu denken.**
