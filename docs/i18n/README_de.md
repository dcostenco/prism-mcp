**Ihr KI-Agent vergisst alles zwischen Sitzungen. Prism behebt das — und lehrt ihn dann zu denken.**

Prism v12.5 ist eine echte **Kognitive Architektur**, inspiriert von der Mechanik des menschlichen Gehirns. Ueber die flache Vektorsuche hinaus bildet Ihr Agent jetzt Prinzipien aus Erfahrung, folgt kausalen Gedankenketten und besitzt das Selbstbewusstsein zu wissen, wann ihm Informationen fehlen. Die gesamte kognitive Pipeline laeuft **100% auf dem Geraet**.

```bash
npx -y prism-mcp-server
```

Funktioniert mit **Claude Desktop · Claude Code · Cursor · Windsurf · Cline · Gemini · Antigravity** — **jeder MCP-Client.**

## Warum Prism?

Jedes Mal, wenn Sie ein neues Gespraech mit einem KI-Codierassistenten beginnen, startet er von Null. Sie erklaeren die Architektur erneut, beschreiben Entscheidungen erneut, listen TODOs erneut auf. Stunden an Kontext — verloren.

**Prism gibt Ihrem Agenten ein Gehirn, das bestehen bleibt — und lehrt ihn dann zu denken.**
