## 왜 Prism인가?

AI 코딩 어시스턴트와 새 대화를 시작할 때마다 처음부터 시작합니다. 아키텍처를 다시 설명하고, 결정을 다시 기술하고, TODO를 다시 나열합니다. 수시간의 컨텍스트 — 사라집니다.

**Prism은 에이전트에게 지속되는 두뇌를 제공하고 — 그런 다음 추론하는 법을 가르칩니다.**
