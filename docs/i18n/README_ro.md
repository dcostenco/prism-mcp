**Agentul tau AI uita totul intre sesiuni. Prism rezolva asta — apoi il invata sa gandeasca.**

Prism v12.5 este o adevarata **Arhitectura Cognitiva** inspirata de mecanica creierului uman. Dincolo de cautarea vectoriala plata, agentul tau formeaza acum principii din experienta, urmareste lanturi cauzale de gandire si are constiinta de sine pentru a sti cand ii lipsesc informatii. Intregul pipeline cognitiv ruleaza **100% pe dispozitiv**.

```bash
npx -y prism-mcp-server
```

## De ce Prism?

De fiecare data cand incepi o conversatie noua cu un asistent de codare AI, acesta porneste de la zero. Re-explici arhitectura, re-descrii deciziile, re-listezi TODO-urile. Ore de context — pierdute.

**Prism ofera agentului tau un creier care persista — apoi il invata sa rationeze.**
