## De ce Prism?

De fiecare dată când începi o conversație nouă cu un asistent de codare AI, acesta pornește de la zero. Re-explici arhitectura, re-descrii deciziile, re-listezi TODO-urile. Ore de context — pierdute.

**Prism oferă agentului tău un creier care persistă — apoi îl învață să raționeze.**
