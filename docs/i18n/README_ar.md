**وكيل الذكاء الاصطناعي الخاص بك ينسى كل شيء بين الجلسات. Prism يصلح ذلك — ثم يعلمه التفكير.**

Prism v12.5 هو **هندسة معرفية** حقيقية مستوحاة من آليات الدماغ البشري. بعيداً عن البحث المتجهي المسطح، يشكل وكيلك الآن مبادئ من التجربة، ويتبع سلاسل سببية من التفكير، ويملك الوعي الذاتي ليعرف متى تنقصه المعلومات. يعمل خط الأنابيب المعرفي بالكامل **100% على الجهاز**.

```bash
npx -y prism-mcp-server
```

يعمل مع **Claude Desktop · Claude Code · Cursor · Windsurf · Cline · Gemini · Antigravity** — **أي عميل MCP.**

## لماذا Prism؟

في كل مرة تبدأ محادثة جديدة مع مساعد برمجة بالذكاء الاصطناعي، يبدأ من الصفر. تعيد شرح البنية، تعيد وصف القرارات، تعيد إدراج المهام. ساعات من السياق — ضائعة.

**Prism يمنح وكيلك دماغاً يستمر — ثم يعلمه التفكير المنطقي.**
