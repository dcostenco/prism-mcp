## 为什么选择Prism？

每次您与AI编程助手开始新对话时，它都从零开始。您重新解释架构，重新描述决策，重新列出TODO。数小时的上下文 — 丢失了。

**Prism给您的代理一个持久的大脑 — 然后教它推理。**
